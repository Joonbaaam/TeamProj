1. 문자열(숫자형식)을 숫자로 바꾸는 메소드

1-1.래퍼클래스.parseXXX(문자열)  :  정적클래스, 반환타입 int

ex: "100"   Integer.parseint()
    "3.14"  Float.parsefloat()
    "3.14"  Double.parsedouble()
1-2.래퍼클래스.valueOf(문자열)      :  반환타입 래퍼클래스
    래퍼클래스.valueOf(기본자료형)  :  반환타입 래퍼클래스


2. 숫자를 문자열로 바꾸는 메소드

2-1. String toString()
     static String toString(숫자);

3. 모든 래퍼클래스는 toString()이 오버라이딩 되어있다.

toString()은 원래 주소가 저장되어있는 데이터를 실제 저장된 문자열데이터로 바꾼다.

4. == :기본 자료형에서는 값을 비교한다. 하지만 참조형에서는 주소를 비교한다.
